# **App Name**: CuisineConnect

## Core Features:

- Restaurant Listing: Display a list of restaurants fetched from the Java-based Restaurant & Delivery Service.
- Menu Display: Show the menu items for a selected restaurant, fetched from the Java backend via the Node.js API gateway.
- User Authentication: Enable user registration and login via the Node.js User & Order Service.
- Order Placement: Allow users to add items to their cart and place orders through the Node.js User & Order Service.
- Order Tracking: Enable users to track their order status, with updates coming from the Java Restaurant & Delivery Service.
- API Gateway: The Node.js backend acts as an API gateway, routing requests from the frontend to the appropriate backend service.

## Style Guidelines:

- Primary color: Vibrant orange (#FF7043), reminiscent of the warmth and energy of a bustling kitchen.
- Background color: Light peach (#FFE7D4), to create a soft and inviting feel.
- Accent color: Mustard yellow (#FFB300), for highlighting key actions.
- Body and headline font: 'PT Sans' for a modern yet friendly user experience.
- Use clear and consistent icons to represent different food categories and actions.
- A clean, grid-based layout will organize restaurant listings and menu items efficiently.
- Subtle transitions and loading animations will provide a smooth user experience.
